Другие теги

Тег <dl>, <dt>, <dd>

Тройка элементов <dl>, <dt>, <dd>, предназначенны для создания списка определений. Каждый такой список начинается с контейнера <dl>, куда входит тег <dt> создающий термин и тег <dd> задающий определение этого термина.

Пример кода:

<dl> <dt>HTML</dt> <dd>(от англ. HyperText Markup Language — &laquo;язык гипертекстовой разметки&raquo;) —стандартизированный язык разметки документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML. </dd> </dl> <dl> <dt>CSS</dt> <dd>CSS (от англ. Cascading Style Sheets — каскадные таблицы стилей) — формальный язык описания внешнего вида документа, написанного с использованием языка разметки. </dd> </dl>

Результат примера:

Тег <pre>

Элемент <pre> определяет блок предварительно форматированного текста. Такой текст отображается обычно моноширинным шрифтом и со всеми пробелами между словами. По умолчанию, любое количество пробелов идущих в коде подряд, на веб-странице показывается как один. Тег <pre> позволяет обойти эту особенность и отображать текст как требуется разработчику.

Пример кода:

<pre> ......oooO............ .....(....)......Oooo. ......)../.....(....). .....(__/.......)../.. ...............(__/... ... Я ТУТ ............ .......... БЫЛ........ </pre>

Результат примера:

            ......oooO............
            .....(....)......Oooo.
            ......)../.....(....).
            .....(__/.......)../..
            ...............(__/...
            ... Я ТУТ ............
            .......... БЫЛ........

Тег <code>

Тег <code> предназначен для отображения одной или нескольких строк текста, который представляет собой программный код. Сюда относятся имена переменных, ключевые слова, тексты функции и т.д. Браузеры обычно отображают содержимое контейнера <code> как моноширинный текст уменьшенного размера.

В отличие от тега <pre> дополнительные пробелы внутри контейнера <code> не учитываются, так же, как и переносы текста. Поэтому используйте тег <br> или <p> для создания переносов.

Пример кода:

<code> <script type="text/javascript"> function count_rabbits() { for(var i=1; i<=3; i++) { alert("Из шляпы достали "+i+" кролика!") } } </script> <input type="button" onclick="count_rabbits()" value="Считать кролей!"/> </code>

Результат примера:

©Нагло украден

Тег <kbd>

Тег <kbd> используется для обозначения текста, который набирается на клавиатуре или для названия клавиш. Браузеры обычно помечают текст в контейнере <kbd> моноширинным шрифтом.

Пример кода:

<p> Control-Alt-Delete (сокращённо <kbd>&lt;Ctrl&gt;+&lt;Alt&gt;+&lt;Del&gt; </kbd> ) — комбинация компьютерных клавиш для IBM-совместимых компьютеров, вызываемая нажатием клавиши <kbd>Delete</kbd> при удерживании клавиш <kbd> Control</kbd> и <kbd>Alt</kbd>: <kbd>&lt;Ctrl&gt;+&lt;Alt&gt;+&lt;Del&gt; </kbd>.Функция комбинации клавиш различна, но обычно она прерывает или облегчает прерывание функции. </p>

Результат примера:

Control-Alt-Delete (сокращённо <Ctrl>+<Alt>+<Del> ) — комбинация компьютерных клавиш для IBM-совместимых компьютеров, вызываемая нажатием клавиши Delete при удерживании клавиш Control и Alt: <Ctrl>+<Alt>+<Del>. Функция комбинации клавиш различна, но обычно она прерывает или облегчает прерывание функции.

Тег <var>

Тег <var> используется для выделения переменных компьютерных программ. Браузеры обычно помечают текст в контейнере <var> курсивным начертанием.

Пример кода:

<p> В данном случае, когда JavaScript встречает слово <var>var</var>, он понимает, что следом указано имя новой переменной. </p>

Результат примера:

В данном случае, когда JavaScript встречает слово var, он понимает, что следом указано имя новой переменной.

Тег <sup>

Тег <sup> отображает шрифт в виде верхнего индекса. Шрифт при этом отображается выше базовой линии текста и уменьшенного размера.

Пример кода:

<p> X<sup>n</sup>+Y<sup>n</sup>=Z<sup>n</sup> </p>

Результат примера:

Xn+Yn=Zn

Тег <sub>

Тег <sub> отображает шрифт в виде нижнего индекса. Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера.

Пример кода:

<p> H<sub>2</sub>O </p>

Результат примера:

H2O

Тег <small>

Тег <small> уменьшает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3. Таким образом, добавление тега <small> уменьшает текст на одну условную единицу. Допускается применение вложенных тегов <small>, при этом размер шрифта будет меньше с каждым вложенным уровнем, но не может быть меньше, чем 1.

Пример кода:

<p> Обычный текст </p> <p><small> Маленький текст </small> </p>

Результат примера:

Обычный текст

Маленький текст

Тег <big>

Тег <big> увеличивает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3. Таким образом, добавление тега <big> увеличивает текст на одну условную единицу. Допускается применение вложенных тегов <big>, при этом размер шрифта будет больше с каждым уровнем.

Пример кода:

<p> Обычный текст </p> <p><big> Большой текст </big> </p>

Результат примера:

Обычный текст

Большой текст

Тег <blockquote>

Тег <blockquote> предназначен для выделения длинных цитат внутри документа. Текст, обозначенный этим тегом, традиционно отображается как выровненный блок с отступами слева и справа (примерно по 40 пикселов), а также с отбивкой сверху и снизу.

Пример кода:

<blockquote> <p> По своей сути рыбатекст является альтернативой традиционному lorem ipsum, который вызывает у некторых людей недоумение при попытках прочитать рыбу текст. В отличии от lorem ipsum, текст рыба на русском языке наполнит любой макет непонятным смыслом и придаст неповторимый колорит советских времен. </p> </blockquote>

Результат примера:

По своей сути рыбатекст является альтернативой традиционному lorem ipsum, который вызывает у некторых людей недоумение при попытках прочитать рыбу текст. В отличии от lorem ipsum, текст рыба на русском языке наполнит любой макет непонятным смыслом и придаст неповторимый колорит советских времен.

Тег <q>

Тег <q> используется для выделения в тексте цитат. Содержимое контейнера автоматически отображается в браузере в кавычках.

Пример кода:

<p> Омар Хайям сказал: <q>Настоящий друг — это человек, который выскажет тебе в глаза все, что о тебе думает, а всем скажет, что ты — замечательный человек.</q> </p>

Результат примера:

Омар Хайям сказал: Настоящий друг — это человек, который выскажет тебе в глаза все, что о тебе думает, а всем скажет, что ты — замечательный человек.

Тег <ins>, <del>

Тег <ins> предназначен для выделения текста, который был добавлен в новую версию документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны.Браузеры обычно помечают текст в контейнере <ins> как подчеркнутый.

Тег <del> используется для выделения текста, который был удален в новой версии документа. Подобное форматирование позволяет отследить, какие изменения в тексте документа были сделаны. Браузеры обычно помечают текст в контейнере <del> как перечеркнутый.

Пример кода:

<p><del>Казнить нельзя</del> <ins>помиловать</ins> </p>

Результат примера:

Казнить нельзя помиловать

Тег <abbr>

Тег <abbr> указывает, что последовательность символов является аббревиатурой. С помощью атрибута title дается расшифровка сокращения, что позволяет понимать аббревиатуру тем людям, которые с ней не знакомы. Кроме того, поисковые системы индексируют полнотекстовый вариант сокращения, что может использоваться для повышения рейтинга документа.

Пример кода:

<p><abbr title="HyperText Transfer Protocol">HTTP</abbr> — протокол прикладного уровня передачи данных изначально — в виде гипертекстовых документов в формате &laquo;HTML&raquo;, в настоящий момент используется для передачи произвольных данных. </p>

Результат примера:

HTTP — протокол прикладного уровня передачи данных изначально — в виде гипертекстовых документов в формате «HTML», в настоящий момент используется для передачи произвольных данных.

Тег <address>

Тег <address> предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д. Планируется, что поисковые системы будут анализировать содержимое этого тега для сбора информации об авторах сайтов.

Пример кода:

<p> <address>Бульвар капуцинов д.1, офис 1</address> </p>

Результат примера:

Бульвар капуцинов д.1, офис 1

Тег <nobr>

Тег <nobr> уведомляет браузер отображать текст без переносов. Если этого тега в коде документа нет, а также имеются переводы строки, они игнорируются и текст выравнивается по левому краю окна браузера или родительского элемента. При этом браузер переводы строк расставляет автоматически, чтобы текст полностью поместился по ширине окна. Использование тега <nobr> заставляет отображать текст без переносов, одной строкой, что может привести к появлению горизонтальной полосы прокрутки. Пользователям придется прокручивать текст по горизонтали, чтобы увидеть его полностью, чего обычно делать никто не любит.

Пример кода:

<p> <nobr>С другой стороны постоянный количественный рост и сфера нашей активности требуют от нас анализа соответствующий условий активизации.</nobr> Повседневная практика показывает, что реализация намеченных плановых заданий играет важную роль в формировании новых предложений. Результат примера:

Повседневная практика показывает, что реализация намеченных плановых заданий играет важную роль в формировании новых предложений. Тег <wbr>

Тег <wbr> указывает браузеру место, где допускается делать перенос строки в тексте, если этого требует ширина родительского элемента.

Пример кода:

<p> <wbr> Гидропневмосистемы </wbr> мобильных и технологических машин </p>

Результат примера:

Гидропневмосистемы мобильных и технологических машин
